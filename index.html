最受欢迎的功能 :

Goto Anything

使用Goto Anything只需几个按键即可打开文件，并立即跳转到符号，行或单词。

快捷键 Ctrl+P⌘+P, 可以使用:

  • 键入文件名的一部分来打开它。
  • 键入 @ 跳转到符号, # 在文件内搜索 以及 : 转到行号。

这些快捷方式可以结合使用， tp@rf可以将您带到文件text_parser.py中的函数read_file。 同样，tp:100会带你到同一个文件的第100行。

转到定义

使用语法定义中的信息，Sublime Text自动生成每个类，方法和函数的项目范围索引。 Goto Definition, 它有三种不同的使用方式：

  • 悬停在符号上时会显示一个弹出窗口
  • 当插入符号时按下 F12
  • Goto Symbol in Project 功能

可以通过配置文件按照语法定制符号索引，使用户可以根据自己的需要定制功能。

多选

同时进行十次更改，而不是一次更改十次。 多重选择允许您一次交互地更改许多行，轻松地重命名变量，并以前所未有的速度处理文件。

尝试按下 Ctrl+Shift+L⇧+⌘+L 将选择分割成行和 Ctrl+D⌘+D 选择下一次出现的选定单词。 用鼠标做多个选择，查看文档 Column Selection.

命令选项板

The Command Palette 保存不常用的功能，如排序，更改语法和更改缩进设置。 只需几个按键，您就可以搜索您想要的内容，而无需浏览菜单或记住模糊的键绑定。

用快捷键 Ctrl+Shift+P⇧+⌘+P开启Command Palette .

强大的API和软件包生态系统

Sublime Text有一个强大的Python API 允许插件增强内置功能。

Package Control 可以通过命令面板进行安装，提供对社区构建的数千个软件包的简单访问。

高度自定义

快捷键绑定，菜单，片段，宏，完成等等 - 几乎所有Subline Text中的内容都可以用简单的JSON文件进行定制。 该系统为您提供了灵活性，因为可以在每个文件类型和每个项目的基础上指定设置。

分割编辑

利用分割编辑支持充分利用宽屏幕显示器。 并排编辑文件，或在一个文件中编辑两个位置。 您可以根据需要编辑尽可能多的行和列。 通过使用多个窗口进行编辑并在每个窗口中使用多个分割来利用多个监视器。

查看View 布局菜单以进行拆分编辑选项。 要将多个视图打开到一个文件中，请使用File 新建视图到文件菜单项。

即时项目开关

Subline Text中的项目捕获工作空间的全部内容，包括修改和未保存的文件。 您可以按照与Goto Anything类似的方式在项目之间切换，并且切换是即时的，没有保存提示 - 所有修改将在下次打开项目时恢复。

高性能

Subline Text由自定义组件构建，提供无与伦比的响应能力。 从强大的定制跨平台UI工具包到无与伦比的语法高亮引擎，Subline Text设置了性能栏。

跨平台

Subline Text可用于Mac，Windows和Linux。 无论您使用何种操作系统，在您拥有的每台计算机上使用Subline Text都需要一个许可证。

Subline Text使用自定义UI工具包，针对速度和美观进行了优化，同时利用每个平台上的本机功能。
